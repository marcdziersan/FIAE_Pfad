# 04 · PostgreSQL

PostgreSQL wird hier mit Schema, Constraints, CTEs, Views, Funktionen und Rollen betrachtet. Die Aufgaben eignen sich als Brücke zwischen Grundlagen-SQL und professioneller Datenbankarbeit.

## Aufgaben

1. Schema mit SERIAL/IDENTITY und Constraints erstellen
2. Beispieldaten einfügen
3. Constraints und Indizes prüfen
4. CTEs, Views und Window Functions verwenden
5. Funktion und Transaktion schreiben
6. Rollen und Backup fachlich einordnen
