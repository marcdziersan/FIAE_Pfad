# 06 · MongoDB

MongoDB wird als dokumentorientierte Datenbank betrachtet. Der Abschnitt zeigt CRUD, Schema-Validierung, Aggregation, Indizes und Import/Export-Grundlagen.

Die Beispiele sind für `mongosh` formuliert.

## Aufgaben

1. Dokumente einfügen und abfragen
2. Dokumente aktualisieren und löschen
3. Schema-Validierung anlegen
4. Aggregation für Reports verwenden
5. Indizes und Suche einordnen
6. Export/Import mit Node.js-Grundidee betrachten
