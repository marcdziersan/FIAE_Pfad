# 02 · SQLite

SQLite eignet sich gut für lokale Lernprojekte, kleine Tools, Tests und Prototypen. Es benötigt keinen Server und kann direkt mit Python geprüft werden.

## Aufgaben

1. Datenbankstruktur erstellen
2. Beispieldaten einfügen
3. CRUD-Abfragen ausführen
4. Views und Reports erstellen
5. Transaktionen und Indizes anwenden
6. Prüfskript ausführen

## Prüfen

```bash
python datenbanken/02_sqlite/06_sqlite_pruefung.py
```
