# 05 · Firebird

Firebird ist eine relationale Datenbank, die in vielen bestehenden Business-Anwendungen vorkommt. Dieser Abschnitt zeigt Grundschema, Sequenzen, Trigger, Stored Procedures und klassische SQL-Auswertungen.

## Aufgaben

1. Tabellen für Kunden und Aufträge erstellen
2. Beispieldaten einfügen
3. Sequenz und Trigger für IDs verwenden
4. Stored Procedure für Reports erstellen
5. CRUD und Transaktionen nachvollziehen
6. Firebird-Besonderheiten fachlich einordnen
