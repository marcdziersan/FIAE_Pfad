# 03 · MySQL / MariaDB

Dieser Abschnitt zeigt typische MySQL-Aufgaben für Webanwendungen: InnoDB, Fremdschlüssel, utf8mb4, Prepared Statements, Indizes und einfache Benutzer-/Backup-Kommandos.

## Aufgaben

1. Schema mit InnoDB und Constraints erstellen
2. Beispieldaten einfügen
3. CRUD mit Prepared Statements in PHP nachvollziehen
4. JOIN-Reports schreiben
5. Transaktionen und Indizes verwenden
6. Benutzer und Export grundsätzlich einordnen
