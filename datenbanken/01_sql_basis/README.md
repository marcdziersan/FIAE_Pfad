# 01 · SQL Basis

Allgemeine SQL-Aufgaben ohne starke Bindung an ein einzelnes Datenbanksystem. Die Beispiele zeigen Grundmuster, die in SQLite, MySQL, PostgreSQL und Firebird mit kleinen Dialektanpassungen wiederverwendbar sind.

## Aufgaben

1. Schema für Kunden und Aufträge entwerfen
2. SELECT, WHERE, ORDER BY anwenden
3. INNER JOIN und LEFT JOIN verstehen
4. GROUP BY und HAVING für Auswertungen nutzen
5. Subqueries und Views einsetzen
6. Transaktionen und Indizes fachlich einordnen
