# HTML Lernpfad

Dieser Abschnitt isoliert HTML. Ziel: semantische Struktur, Navigation, Formulare, Tabellen, Medien, Barrierefreiheit und prüfbare Dokumentstruktur.

## Beispiele

1. `01_semantik` – Header, Main, Section, Article, Footer
2. `02_formulare` – Labels, Inputs, Fieldsets, Pflichtfelder
3. `03_tabellen` – Tabellen mit Caption, Thead, Tbody, Tfoot
4. `04_barrierearme_struktur` – Skip-Link, Landmarken, sinnvolle Überschriften

Alle Beispiele sind reine HTML-Dateien und ohne Build-Prozess prüfbar.
