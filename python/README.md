# Python Lernpfad

Dieser Abschnitt isoliert Python als Skript- und CLI-Sprache. Ziel: Syntax, Funktionen, Dateien, JSON, einfache Fachlogik und Tests.

## Reihenfolge

1. `01_hello_world.py`
2. `02_datentypen_funktionen.py`
3. `03_taschenrechner_cli.py`
4. `04_telefonbuch_json.py`
5. `05_business_cli.py`
6. `06_tests/test_business.py`

## Start

```bash
python python/01_hello_world.py
python python/03_taschenrechner_cli.py 10 + 5
python python/04_telefonbuch_json.py add Marcus marcus@example.de 02306-0000
python -m unittest discover -s python/06_tests
```
