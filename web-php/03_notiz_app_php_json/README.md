# 03 Notiz-App mit PHP und JSON

Kleines CRUD-Projekt mit PHP, HTML, CSS, JavaScript und JSON-Datei.

## Start

```bash
php -S localhost:8000 -t public
```

## Funktionen

- Notiz anlegen
- Notizen anzeigen
- Notiz als erledigt markieren
- Notiz löschen
- serverseitige Validierung
- JSON-Datei als einfacher Speicher
