# CSS Lernpfad

Dieser Abschnitt isoliert CSS. Ziel: Selektoren, Box-Modell, Layout, Responsive Design, Grid/Flexbox, Zustände, Animation und Druckansicht.

## Beispiele

1. `01_box_model` – Margin, Padding, Border, Width
2. `02_flexbox_navigation` – flexible Navigation und Cards
3. `03_grid_responsive` – responsive Projektkarten
4. `04_animation_states` – Hover, Focus, Transition, Keyframes
5. `05_print_styles` – druckbare Ansicht
