# Lehrplan · FIAE Grundlagenpfad

## Ziel

Der Lernpfad zeigt Grundlagen der Anwendungsentwicklung in isolierten Sprachpfaden und führt sie am Ende in einer kleinen Gesamtanwendung zusammen.

## Phase 1 · Einzelne Technologien

### HTML

- Dokumentstruktur
- Semantik
- Navigation
- Formulare
- Tabellen
- Barrierearme Struktur

### CSS

- Box-Modell
- Selektoren
- Flexbox
- Grid
- Responsive Design
- Zustände und Animation
- Druckansicht

### JavaScript

- Script-Einbindung
- Variablen und Funktionen
- DOM-Zugriff
- Events
- Validierung
- localStorage
- Fetch

### PHP

- Ausgabe und Variablen
- Arrays und Funktionen
- Formulare und Validierung
- JSON-Dateispeicherung
- OOP/Repository
- Mini-API

### Python

- CLI-Ausgabe
- Funktionen
- Listen und Dictionaries
- JSON-Dateien
- Telefonbuch
- Business-Logik
- Unittest


### Datenbanken

- SQL-Grundlagen
- SQLite lokal ohne Server
- MySQL/MariaDB für Webanwendungen
- PostgreSQL mit CTEs, Views und Funktionen
- Firebird als relationale Business-Datenbank
- MongoDB als dokumentorientierte Datenbank
- CRUD, JOIN, Aggregation, Indizes, Transaktionen und Reports

## Phase 3 · Kombinierte Prüfung

`99_kombi_pruefung` verbindet:

- HTML für Struktur
- CSS für Layout und Druck
- JavaScript für Bedienung und Fetch
- PHP für API und Datenvalidierung
- Python für Datenprüfung und Report
- Datenbankgrundlagen als eigener Nachweispfad

## Prüfergebnis

Am Ende ist sichtbar, dass die Grundlagen nicht nur einzeln verstanden, sondern auch in einer einfachen Gesamtanwendung zusammengesetzt werden können.
