# JavaScript Lernpfad

Dieser Abschnitt isoliert JavaScript im Browser. Ziel: Syntax, DOM, Events, Validierung, localStorage, Module-Denken und Fetch-Verständnis.

## Reihenfolge

1. `01_hello_world` – Script-Einbindung und Ausgabe
2. `02_variablen_funktionen_arrays` – Datentypen, Arrays, Funktionen
3. `03_dom_events` – DOM-Auswahl, Events, Klassen ändern
4. `04_taschenrechner` – Eingaben, Operatoren, Fehlerfälle
5. `05_notiz_app_localstorage` – CRUD im Browser mit localStorage
6. `06_fetch_demo` – Fetch gegen lokale JSON-Datei

Alle Beispiele laufen direkt im Browser über `index.html`.
